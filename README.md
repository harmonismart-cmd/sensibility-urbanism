# Sensibility Urbanism
### Measuring the Invisible Qualities of Cities
Paper:
https://harmonismart-cmd.github.io/sensibility-urbanism/

Sensibility Urbanism is an urban theory proposed by Kazuto Kojima, a producer and artist based in Osaka, Japan.

This framework explores how cities can be understood through sensory experience, atmosphere, and qualitative perception, rather than only through quantitative indicators such as population, economic growth, or infrastructure.

## Core Concept

Sensibility Urbanism focuses on qualitative urban values such as:

- Atmosphere
- Perception
- Emotional memory
- Bodily experience
- Everyday urban life

Example indicators include:

- Sunset Staying Rate
- Detour Impulse Count
- Walking Drift Distance
- Sound Recognition Diversity
- Smell Awareness Rate
